üìö Usage Guide

When to use SCWP (Default):

  • New projects and fresh designs
  • When you want modern, flexible styling
  • Projects using custom branding
  • When hover effects are desired

When to use CCM Variants:

  • WordPress admin consistency
  • Porting from CCM plugin
  • Enhanced form focus states needed
  • Official WordPress admin look

Best Practices:

  • Mix both styles as needed
  • Use CCM for admin forms
  • Use SCWP for public-facing UI
  • Maintain consistency within pages
