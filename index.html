1Download Files

Include the CSS files in your WordPress plugin or theme:

HTML Basic Installation 
<link rel="stylesheet" href="path/to/scwp-admin-ui.css">

Or use the modular approach:

HTML Modular Installation 
<link rel="stylesheet" href="path/to/scwp-admin-ui-variables.css">
<link rel="stylesheet" href="path/to/scwp-admin-ui-components.css">
<link rel="stylesheet" href="path/to/scwp-admin-ui-utilities.css">

2WordPress Plugin Integration

Enqueue the styles in your plugin:

PHP WordPress Enqueue 
function enqueue_scwp_admin_ui() {
    wp_enqueue_style(
        'scwp-admin-ui',
        plugin_dir_url(__FILE__) . 'assets/css/scwp-admin-ui.css',
        array(),
        '1.0.0'
    );
}
add_action('admin_enqueue_scripts', 'enqueue_scwp_admin_ui');

3Add JavaScript (Optional)

Include the JavaScript for interactive components:

PHP JavaScript Enqueue 
function enqueue_scwp_admin_ui_js() {
    wp_enqueue_script(
        'scwp-admin-ui',
        plugin_dir_url(__FILE__) . 'assets/js/scwp-admin-ui.js',
        array(),
        '1.0.0',
        true
    );
}
add_action('admin_enqueue_scripts', 'enqueue_scwp_admin_ui_js');
